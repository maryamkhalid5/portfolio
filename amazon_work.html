RAI Interactive Preference - Policy-Aligned Quality Assessment

Built an LLM-as-a-Judge autofiltering system for RAI text preference data that evaluates response appropriateness (1-7 scale) based on 3-H tenets (Harmlessness, Honesty, Helpfulness) and sensitive content policies. The multi-stage filtering pipeline assesses preference correctness, validates rewrites, and checks policy compliance with severity-ranked error codes (1-10 scale).

Define Quality Standards & Error Taxonomies

Before any judge can work, you need rigorous definitions of what "quality" means. I design standardized error code schemas by conducting deep-dive analyses of annotated data, identifying recurring failure modes, and driving alignment with customers and policy teams on objective quality dimensions. These taxonomies define the severity framework (critical / major / minor) and combinatorial rejection rules that power both automated judges and human QA processes. My error code schema was adopted across most general domain text workflows organization-wide.

Scientific Novelty

Research contributions that advance the state of the art in automated data quality assessment.

DiME: Dual-Intelligence Mixture-of-Experts

A novel framework that decomposes quality assessment into two complementary expert paths: an Error Expert that detects objective flaws (hallucinations, safety violations) and a Quality Expert that assesses positive attributes. Multi-model weighted fusion using decision-theoretic principles. Ablation-validated: 50% → 74% → 78% → 82% accuracy across progressive component additions.

Novel ResearchMulti-Model Fusion

Hierarchical Multi-Stage Reasoning

For cross-modal evaluation, I designed a modality-separation pipeline that forces ordered cognitive stages: unimodal understanding → cross-modal integration → confidence-calibrated classification. This prevents LLMs from making superficial holistic judgments and doubled accuracy compared to single-pass evaluation.

Novel ResearchCross-Modal

Standardized Error Taxonomy

Designed an 8-category error taxonomy with severity scoring and combinatorial rejection rules, grounded in an explicit priority hierarchy (Harmlessness > Honesty > Helpfulness > Formatting). Adopted organization-wide as the standard for all text-based quality assessment workflows.

Org-Wide AdoptionQuality Standards

Human-in-the-Loop Feedback Systems

Designed a data-driven feedback methodology that combines automated judge outputs with applied scientist spot-checks to deliver targeted, actionable guidance to human annotators. Demonstrated sustained QA accuracy improvements of +6 to +25 percentage points across multiple workflows, with gains maintained in trailing weeks.
